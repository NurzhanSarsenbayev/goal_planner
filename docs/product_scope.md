# Product Scope

## Working name

Goal Planner

## Product idea

A mobile daily planner that connects long-term goals with daily actions.

The app helps users:

- define long-term goals;
- break goals into concrete tasks;
- schedule tasks into Today;
- complete tasks;
- see which goals moved forward.

## Core user problem

Most planners separate daily tasks from long-term goals.

Users can create goals, but their daily task list often becomes a disconnected mix of chores, habits, errands, and long-term work.

This product focuses on one core idea:

> Every important daily task should be connected to a goal or clear life direction.

## Target user for MVP

The first target user is a person who already uses planners, habit trackers, notes, or calendars, but feels that they are fragmented or not connected to real progress.

Initial validation user: one real user from the household.

Later validation: 3-5 external users with similar planning habits.

## MVP positioning

Not an all-in-one life management app.

The MVP is a goal-linked daily planner:

> Goals -> Tasks -> Today -> Completion -> Progress report

## In MVP

- Today screen
- Goals screen
- Goal details screen
- Tasks linked to goals
- Tasks scheduled for today
- Task completion
- Simple goal progress
- Basic daily report

## Out of MVP

- Backend
- Authentication
- Cloud sync
- AI goal decomposition
- Full node/graph editor
- Weight tracker
- Expense tracker
- Google Calendar integration
- Team/shared planning
- Complex recurring rules
- Full gamification
- Web app
- Desktop app

## Product principle

If a feature does not improve the core loop:

> create goal -> break into task -> schedule today -> complete -> see progress

then it does not belong in the MVP.
